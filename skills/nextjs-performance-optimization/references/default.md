# Default

Classify the request as `design`, `implement`, `review`, or `migrate`, then
resolve that explicit task before changing a performance-sensitive Next.js data
surface. If the task remains ambiguous, inspect the current data path and ask
only for the product choice that changes pagination, visibility, or API
compatibility.
